# Minecraft Resource Hub (Fix Lag & Litematica)

Website chia sẻ tài nguyên Minecraft giao diện đẹp mắt, bắt mắt, tối ưu hóa cho Vercel.

## Cấu trúc thư mục (Phân công công việc):
- `index.html`: Khung cấu trúc trang web.
- `style.css`: Thiết kế giao diện, màu sắc nổi bật, hiệu ứng glow gaming.
- `data.js`: Chứa danh sách các file (Dễ dàng thêm/sửa file Fix Lag & Litematica + Link rút gọn + Link ảnh).
- `script.js`: Logic chuyển tab và mở link rút gọn trực tiếp.
- `vercel.json`: Cấu hình cho Vercel.

## Hướng dẫn đưa lên Vercel:
1. Đăng nhập vào [Vercel.com](https://vercel.com).
2. Tải toàn bộ thư mục này lên GitHub hoặc Kéo thả (Drag & Drop) trực tiếp vào giao diện Vercel New Project.
3. Bấm Deploy là trang web sẽ chạy online ngay lập tức!
